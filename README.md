# 🚀 ReqRes API Automation Testing

Automation testing project menggunakan **Java + RestAssured + TestNG** untuk menguji endpoint dari https://reqres.in.

Project ini dibuat untuk practice dan portfolio API Automation Testing.

---

## 📦 Tech Stack

- Java 17+
- RestAssured
- TestNG
- Gradle
- SLF4J (logging)
- ReqRes Public API
